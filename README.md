# スタンプラリー



## 1. チェックポイントの登録/登録解除
/function stamprally/tool で入手  

### ツール使い方
- ブロックに対して右クリックで登録(数値、テキスト、説明を入力 -> ロードマップ用)  
- ブロックに対して左クリックで登録解除  
- スニークしながら右クリックで一覧表示・全登録解除  



## 2. スタンプボードの入手
/function stamprally/board で入手  



## 3. スタンプボードの初期化
- スタンプボードはアイテム依存ではなく、プレイヤー依存  
/function stamprally/reset で初期化  



## 4. スタンプボードの使い方
### スタンプを押す
- 手に持ち、登録したチェックポイントに右クリックすることでスタンプを押す  

### スタンプラリーのロードマップを表示
- 手に持ち、スニークしながら右クリックすることでロードマップを表示  



## 5. コマンド側からの参照用
> これらはコマンド側からの参照用のため、コマンド側から操作することはできません  

- Score  
  - sr_count: 0~13 スタンプを押している数, すべて終わっている場合13  

- Tag  
  - sr_stamp_[Number]: どこのスタンプをおしてるかどうか, Numberには 1~13の数値が入る
